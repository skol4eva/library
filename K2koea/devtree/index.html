class 파일명
  • common.css : 공통 사용항목(기본테그 정리, 간격, 웹폰트항목, 폰트 사이즈 등)
  • layout.css : 기본 layout에 포함되는 항목들 (header, footer, gnb등)
  • popup.css : 윈도우 popup에 사용되는 css (레이어팝업 포함)
  • main.css
  • contents.css : 컨텐츠에서 사용하는 css import
class,id 기본규칙
  • class와 id 작성 시 카멜기법 사용하지 않고 "_"로 단어와 단어를 연결한다.
  • word_word 방식을 사용하고 대문자는 사용하지 않는다.
  • 숫자를 앞에 넣지 않는다
  • class명칭은 해당 항목의 영문이름으로 앞 1글자, 중간 1글자, 끝 1글자를 따서 약자로 줄인다. (예>table : .tbl, contents : .cnt)
  • class 작성 순서

    position 등 위치, z-index, background 등 배경, width, height, margin, padding, font style등
id규칙 id값은 js에서 필수로 처리되지 않는 한 개발과 협의 후 사용할것
class규칙
  • logo : .lg
  • gnb : .gnb_wrap > .gnb > .gnb_
  • mlb : .mlb_wrap > .mlb > .mlb_
  • body contents 부분 : .cnt_wrap > .cnt > ...
  • table : .tbl_wrap> .tbl1, tbl2, tbl3,...
  • btn : .btn_wrap> .btn1, .btn2, .btn3
  • layer : .lypop_wrap> .lypop >...
  • popup : .pop_wrap > .pop...
  • footer : .ft_wrap > .ft
